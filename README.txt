This only works locally. i.e on LAN.
This lets you host your files(txt,jpg,mp3,js,etc.) on your machine, all you need is Node

-Download the repository

-npm install

-node main

-copy the files you want to host in this repository

thats it!

Now you can download the hosted files using the Ip of the computer on which it is hosted and port no. 3000

Just visit <<IP Address>>:3000 on your browser.Here Ip is the local Ip of machine on which it is hosted.
